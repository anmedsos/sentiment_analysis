Repository ini berisi kumpulan dataset berupa text tweet yang telah disertai label sentimen. 
Dataset tersebut yang dihasilkan dalam tugas perkuliahan Analitika Media Sosial, 
Program Magister Teknologi Informasi, Universitas Indonesia.
Anda diperkenankan menggunakan kembali dataset tersebut untuk penelitian atau kebutuhan akademik lainnya.
Silahkan menuliskan citation pada hasil pekerjaan yang menggunakan dataset dalam repository ini.

1. Muhammad, A Nizar.,  Fathin, R Muhammad. Waste Management yang Diterapkan Pemerintah. 2023. Universitas Indonesia
2. Fauzi, Ahmad., Butar Butar, Jayadi. Entity Sentiment Analysis Untuk Prediksi Dukungan Terhadap Calon Presiden. 2023. Universitas Indonesia
3 Lorenzo, Joash., Sudar, LZ Sabrina. Sentiment Analysis terhadap Polisi pada Sosial Media Twitter. 2023. Universitas Indonesia
4. Purnama, A Anton. Sentiment Analisis terhadap Sumur Resapan Jakarta. 2023. Universitas Indonesia
5. Andana, F Qodri. Sentiment Analysis tentang Upah Buruh. 2023. Universitas Indonesia
